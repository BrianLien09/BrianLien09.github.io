PA: A Very Cool Calculator

題目概述

計算折扣後的價格。

解題思路

  1. 輸入折扣百分比(k)和原價(w)
  2. 計算折扣後價格：(k / 100.0) * w
  3. 使用浮點數確保計算精確度

程式碼重點

  • 使用 long 處理可能的大數輸入
  • 使用浮點數運算確保精確度

完整程式碼

PB: Bar

題目概述

根據輸入的三個數字組合輸出對應的句子。

解題思路

  1. 輸入三個數字 a, b, c
  2. 根據每個數字選擇對應的字串：
    • a: 形容詞 (1:handsome, 2:rich, 3:smart)
    • b: 名字 (1:Allcky, 2:Jimmy, 3:Chen)
    • c: 動作 (1:dances, 2:eats, 3:writes)
  3. 將三個字串組合輸出

程式碼重點

使用 if-else 結構進行字串映射，確保每個數字都有對應的輸出。

完整程式碼

PC: Jims Magical Coffee Spill I

題目概述

計算連續喝咖啡的最長時間。

解題思路

  1. 記錄當前的最大時間點 (stop)
  2. 對每次喝咖啡：
    • 讀入開始時間 p 和持續時間 v
    • 更新最大時間點：stop = Math.max(stop, p) + v
  3. 輸出最終的時間點

程式碼重點

  • 使用 Math.max 確保時間連續性
  • 考慮之前的醉酒時間是否已結束

完整程式碼

PE: Shot Eagle Eye

題目概述

找出包含所有1的最小矩形面積。

解題思路

  1. 讀取矩陣大小 r x c 和矩陣內容
  2. 找出所有1的最大最小行列座標
  3. 計算最小矩形面積

程式碼重點

  • 使用四個變數記錄1出現的範圍
  • 面積計算：(maxr - minr + 1) * (maxc - minc + 1)

完整程式碼

PF: What is the day of week

題目概述

給定起始日和經過的天數，計算最後是星期幾。

解題思路

  1. 讀取起始日(day)
  2. 讀取所有數字並加總(N)
  3. 計算 (N + day) % 7 決定星期幾

程式碼重點

  • 使用模運算處理循環天數
  • 使用 if-else 輸出對應的星期名稱

完整程式碼

PG: The Definition of Doom

題目概述

根據四個分數判斷是否達到標準。

解題思路

  1. 讀取四個分數
  2. 計算：
    • 總和是否 >= 350
    • 最高分和最低分差 <= 25
  3. 同時滿足則輸出 “In”，否則 “Nah”

程式碼重點

  • 使用巢狀 Math.max/min 找出最值
  • 使用 -1 作為結束條件

完整程式碼

PH: Stock Baron

題目概述

計算股票交易的最大獲利。

解題思路

  1. 讀取價格序列
  2. 維護目前看到的最低價格
  3. 對每個價格計算：
    • 如果當前價格低於最低價，更新最低價
    • 否則計算可能的獲利，更新最大獲利

程式碼重點

  • 一次遍歷完成計算
  • 動態更新最低價格和最大獲利

完整程式碼

  • 一次遍歷完成計算
  • 動態更新最低價格和最大獲利
